# Art source

Editable art (Aseprite, AI, PSD, etc.). **Never** load these from game code.

Export game-ready PNGs into the matching folder under `assets/`.

Layout mirrors `assets/`:

```text
art_source/
├── characters/player/
├── enemies/
├── spells/
├── buffs/
├── ui/
├── maps/
├── vfx/
└── audio/
```

See `assets/README.md` and `docs/TECH_STACK.md`.
